Ducky Script is a scripting language used to automate keystrokes on a computer. It was created by the creators of the USB Rubber Ducky, a popular hacking tool. Ducky Script is simple and easy to use, and allows users to create macros that can be executed on any computer running Windows, macOS, or Linux. The language is based on a series of commands that simulate keyboard input, allowing users to automate tasks such as opening applications, running scripts, and entering login credentials.

Ducky Script has gained popularity among security professionals and enthusiasts as a tool for automating penetration testing and other security-related tasks.

What can it do?

Ducky Scrip can do many things including:

  • Automate keystrokes on a computer
  • Create macros
  • Perform repetitive tasks
  • Execute security-related tasks

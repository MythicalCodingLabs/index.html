One of the most powerful features of Ducky Script is the REM function, which is used to add comments to your script.

Comments can be used to explain what your code does, add notes, or provide reminders.

This functin is very usefull in toturial pecies of code as it can explain the code to someone with less experince then the coder. It can also be usefull in long and complicated functions so the user can know what each section.

To add it into your code you simply add a new line and type REM the comment you can replace the comment with anything you want.

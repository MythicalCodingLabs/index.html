REPEAT

index.html Ducky Script

The Ducky Script REPEAT function is used to repeat code in Ducky Script. To use this function:
1. First, you create the function by writing REPEAT.
2. Then, you specify how many times you want the code to be repeated by writing REPEAT [number of times].
3. Finally, you write the code that you want to be repeated.

For example, if you want to repeat a code block 3 times, you would write:
REPEAT 3
[your code block here]
This will repeat the code block 3 times and then continue on to the next code.

DELAY

index.html Ducky Script

The Ducky Script DELAY function is used to slow down the execution of commands in Ducky Script. This can be useful in situations where you need to wait for a certain action to complete before executing the next command. To use the DELAY function, you first create the function like this:
DELAY

Then you specify the number of milliseconds you want to delay, like this:
DELAY 1000

This will pause the script for one second (1000 milliseconds). You can adjust the delay time as needed by changing the number of milliseconds specified after the DELAY command. For example, to wait for two seconds, you would use DELAY 2000.

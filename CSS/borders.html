CSS Border Property

index.html's CSS

The CSS border property is used to set the border around an HTML element. The border property can be used to set the width, style, and color of the border. The syntax of the border property is as follows:
border: [width] [style] [color];
For example, if you want to set a red border with a width of 2 pixels and a dashed style, you would write:
border: 2px dashed red;
This will create a border around the HTML element with a width of 2 pixels, a dashed style, and a red color.

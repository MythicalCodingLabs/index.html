HTML Headers

index.html HTML

Headers in HTML are defined using different tags, such as <h1>, <h2>, <h3>, <h4>, <h5>, and <h6>. The header tag is used to define the headings in a document. The <h1> tag is used for the most important heading, while <h6> is used for the least important.

Example

<h1>This is an h1 header</h1>
<h2>This is an h2 header</h2>
<h3>This is an h3 header</h3>
This will display the headers as follows:

This is an h1 header

This is an h2 header

This is an h3 header

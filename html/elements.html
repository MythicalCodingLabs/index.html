HTML Elements

index.html HTML

An HTML element is a type of HTML document component, one of several types of HTML nodes. They are used to add visuals to the webpage. You open an element using a tag with the elements name in it (You can find the elements under our html navigation tab): <tagname> and then you close the element with </tagname>.

There are hundreds of tags that each create thier own element so be sure that your using the right one.

You can also add classes to tags. A class will link it to an element in CSS so you can edit the style of the element(change size, color, rounded edges, ect.). That will be coverd in the CSS section.

You can also give elements id's so you can interface with your element in JavaScript. You add this using id="yourIdHere". This can be used with the JavaScript function const yourElement = document.getElementById('yourIdHere'). This can be used to change the code in this div and many other things!

Example

In this example we will be creating a simple div system using the <div> tag:
<div>Hello world</div>

This will create a new div element that contains the text hello world.

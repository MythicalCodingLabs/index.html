Index.html HTML

The body tag contains all of the content that is visible on your webpage. Text can be added using the <p> tag, which creates a new paragraph. To add an image, use the <img> tag, and specify the image source (src) and alternative text (alt) as attributes.

Lists

Lists are created using the <ul> and <li> tags. <ul> creates an unordered list, while <ol> creates an ordered list. Each list item is defined with the <li> tag.

Links

Links can be created using the <a> tag. To create a link, specify the URL you want to link to in the href attribute, and the text that should be displayed as the link in between the opening and closing <a> tags.

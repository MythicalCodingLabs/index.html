Images

index.html HTML

Images can be added using the <img> Tag. This tag does not need to be closed. It is used by first adding your images path or link. If the image is stored in the websites files then just add the path usingsrc="Path here". It will look like this <img src="main/image.png">. Remember that you will need to change the file paths according to your site.

If your image is stored on a link you will link it in the same way, but instead of a path it will be your link. That will look like <img src="www.imagehost.com/image.png">. Replace the link with your images link.

To link in html use the <a> Tag. Using the <a> tag is simple. Just add <a herf="www.yoursite.com">Display as link</a>. Replace www.yoursite.com with the site that your linking it to and Display as link to what you want to be linked.

Example

<a herf="www.indexhtml.dev">Index HTML</a>
This will display this link Index HTML and if you click on it it will direct you to www.indexhtml.dev.
